History

  1. javaScript was released with the initial need for a "glue language", that was easy for Web-designers and part-time programmers to assemble components like images and plugins.
  2. It was initially named LiveScript, but later renamed to JavaScript when officially deployed in the 2.0 beta.
  3. In December 1995, very soon after the realse of JavaScript for browsers, an implementation was offered for server-side scriptingss
  4. Since then many different server-side JS implementations have been introduced. NOde.js being one of the most popular in 2009
  5. As with most scripting languages, JavaScript is dynamically typed; a type is associated with each value, rather than just with each expression.
  6. JavaScript uses prototypes where many other object-oriented languages use classes for inheritance.
