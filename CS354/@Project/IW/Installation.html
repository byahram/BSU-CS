Installation

There are dozens of ways to install and run your own JS programs, Pretty much any IDE is going to have a interpreter that is built in or has extensions for.

My personal favourite way to run JS is with Visual studio code, which is the way i will explain Here.

  1. Install Visual Studio Code


  2. Install and download Code Runner from extensions tab on the left.


  3. Write you javaScript program


  4. Do the command Ctr+Shift+P and then select run whenever you want to run the program


  5. To run your JS code on a web server if using it along with HTML do the folowing steps:


  6. Run command Ctr+Shift+P, and then click on 'Control Task Runner'


  7. replace your tasks.json File with the correct format for your operating System


  8. Use Ctr+Shift+B to open up your webpage
